# 🛒 買い物リスト

スーパーの売り場順で買い物メモを整理できる、シングルファイル構成の Web アプリです。  
入力した買い物リストをカテゴリ分けし、チェックしながら効率よく買い回りできます。

## 特徴

- テキストを貼り付けるだけでリストを作成（`、` / `,` / 改行に対応）
- デフォルトカテゴリ（肉・魚介、野菜、乳製品など）へ自動分類
- 未分類アイテムをその場で「キーワード登録」して次回以降は自動分類
- カスタムカテゴリの追加・削除に対応
- 売り場順（カテゴリ順）の並び替え設定
- 進捗表示（完了数、カテゴリごとの達成率、全完了時メッセージ）
- 設定は `localStorage` に保存され、ブラウザ再訪時に復元

## 画面フロー

1. `📋 リストを貼り付け`
2. `🏪 売り場別に並べる`
3. チェックしながら買い物
4. 未分類があれば `＋登録` でキーワード学習
5. `⚙️` から売り場順を調整

## 使い方

### 1) ローカルで開く

このリポジトリはビルド不要です。`index.html` をブラウザで開くだけで動作します。

```bash
git clone https://github.com/seapachi/glrocery_list.git
cd glrocery_list
# index.html をブラウザで開く
```

### 2) 入力例

```text
R-1 6個、そば5個、グミ、納豆、鶏もも肉、キャベツ
```

## 保存データ

`localStorage` の以下キーを使用します。

| Key | 内容 |
| --- | --- |
| `grocery-custom-keywords` | キーワード -> カテゴリの対応 |
| `grocery-custom-categories` | ユーザー作成カテゴリ一覧 |
| `grocery-category-order` | 売り場順（表示順） |

## 技術構成

- HTML + CSS + JavaScript（単一ファイル）
- React 18（UMD / CDN）
- ReactDOM 18（UMD / CDN）
- Babel Standalone（ブラウザ内変換）
