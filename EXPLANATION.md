# EXPLANATION

## このアプリの目的
このアプリは、買い物メモをスーパーの売り場順に整理して、買い忘れや往復を減らすためのツールです。

## できること
- 文字列リストを貼り付けるとカテゴリごとに自動で分類します。
- チェックボックスで購入済みを管理できます。
- 未分類アイテムはその場でキーワード登録でき、次回から自動分類されます。
- カスタムカテゴリの作成・削除ができます。
- 設定画面でカテゴリ順（売り場順）を変更できます。

## デフォルトカテゴリの管理
- デフォルトカテゴリは `default-categories.json` で管理します。
- 開発者は `index.html` を編集せずに、JSONを編集するだけでカテゴリを追加できます。
- アプリ起動時にJSONを検証し、不正な場合は起動エラー画面を表示して停止します。

## ユーザー設定の保存と共有
- ユーザーが追加したキーワード・カテゴリ・並び順は `localStorage` に保存されます。
- 設定画面から JSON エクスポート/インポートが可能です。
- インポート時に衝突がある場合は、一覧で「現在の値」か「インポート値」を選んで一括反映します。
- 不正なインポートファイルは全体中止し、既存データは変更しません。

## 起動の前提
`default-categories.json` を `fetch` で読み込むため、`file://` 直開きではなくローカルサーバーで起動します。

例:

```bash
python -m http.server 8000
```

## 構成
- `index.html`: UIとアプリ本体ロジック
- `default-categories.json`: デフォルトカテゴリ定義
- `README.md`: 使い方と仕様
- `PLAN.md`: このリポジトリでの進捗管理
